Technologies and Tools for Project

Programming Language:
- Python (primary)

Datasets:
- Google Cluster Trace 2019 (BigQuery)
- Azure Public Dataset V2 (2019)
- Alibaba Cluster Trace 2018 (optional extension)

Simulation & Orchestration:
- SimPy (Python discrete-event simulator) for custom scheduling environment
- CloudSim Plus (optional, Java-based) if richer simulation needed
- Kubernetes concepts for placement policies (affinity, taints, tolerations)

Machine Learning / Deep Learning:
- PyTorch (primary)
- scikit-learn
- Prophet / ARIMA (time-series forecasting)

Reinforcement Learning:
- Stable-Baselines3 (PPO, DQN, SAC)
- OpenAI Gym (custom environment wrapping simulator)
- Ray RLlib (optional, for scaling experiments)

Visualization & Experiment Tracking:
- Matplotlib / Plotly
- Pandas
- Weights & Biases (W&B) or MLflow for experiment tracking

Reproducibility:
- Docker for environment consistency
- GitHub/GitLab for version control
